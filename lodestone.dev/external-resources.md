# External Resources

Community members have developed the following resources, tools and packages; these are not maintained under the **lodestone** project:

- [@davispuh](https://github.com/davispuh) has published an Arch package, `lodestone-git`, [in the AUR repository](https://aur.archlinux.org/packages/lodestone-git).
- [@DrFrankensteinUK](https://github.com/DrFrankensteinUK) has written [a **lodestone** on Synology NAS guide](https://drfrankenstein.co.uk/lodestone-in-container-manager-on-a-synology-nas/).
- [@DyonR](https://github.com/DyonR) has developed [magnetico2lodestone](https://github.com/DyonR/magnetico2lodestone), a collection of scripts for importing into **lodestone** from Magnetico and other sources.
- [@DyonR](https://github.com/DyonR) has written [a **lodestone** on Unraid guide](https://github.com/DyonR/lodestone-unraid).
- There is a project initiative that simplifies the setup of Proxmox Virtual Environment (VE), they provide a [script to set up **lodestone** in an LXC containers](https://community-scripts.github.io/ProxmoxVE/scripts?id=lodestone)
